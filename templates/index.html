05-893 Interactive Data Science Fusion Table

Jieli Zhou

These two fusion tables visualize the geolocations of crime reports for different cities from January to June 2017 from FBI. Clicking on the each location dot will display detailed information about the city, like counts for different types of crimes. From theses maps we can easily answer the question which city is safer and back our answers with the corresponding stats. Also we can see that the number of aggravated assult in one city roughly coincides with the number of violent crimes in that city.
